O que é JavaScript e como adicioná-lo ao HTML

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. É uma das três tecnologias principais da web, junto com HTML e CSS.

Como adicionar JavaScript ao HTML:

<!-- Método 1: Inline --> <button onclick="alert('Olá Mundo!')">Clique aqui</button> <!-- Método 2: Interno --> <script> function saudacao() { alert('Olá Mundo!'); } </script> <!-- Método 3: Externo (recomendado) --> <script src="script.js"></script>

Variáveis

Variáveis são containers para armazenar dados. Em JavaScript, podemos declarar variáveis usando var, let ou const.

// Declaração de variáveis var nome = "João"; // Forma antiga (escopo de função) let idade = 25; // Forma moderna (escopo de bloco) const PI = 3.14159; // Constante (não pode ser reatribuída) // Reatribuição de valores idade = 26; // Válido // PI = 3.14; // Erro! Constante não pode ser reatribuída

Tipos de Dados

JavaScript tem tipos de dados dinâmicos, o que significa que uma mesma variável pode conter diferentes tipos de dados.

// Tipos primitivos let texto = "Olá Mundo"; // String let numero = 42; // Number let decimal = 3.14; // Number let booleano = true; // Boolean let nulo = null; // Null let indefinido = undefined; // Undefined let simbolo = Symbol('id'); // Symbol (ES6) // Tipo objeto let pessoa = { // Object nome: "Maria", idade: 30 }; let frutas = ["maçã", "banana", "laranja"]; // Array

Operadores e Condicionais

Operadores permitem realizar operações em variáveis e valores. Condicionais permitem executar código baseado em condições.

Operadores:

// Aritméticos let soma = 10 + 5; // 15 let subtracao = 10 - 5; // 5 let multiplicacao = 10 * 5; // 50 let divisao = 10 / 5; // 2 let resto = 10 % 3; // 1 // Comparação let igual = 5 == "5"; // true (valor igual) let estritamenteIgual = 5 === "5"; // false (valor e tipo iguais) let diferente = 5 != "5"; // false let maiorQue = 10 > 5; // true // Lógicos let e = (5 > 3) && (10 > 5); // true let ou = (5 > 10) || (10 > 5); // true

Condicionais:

// if/else let hora = 14; if (hora < 12) { console.log("Bom dia!"); } else if (hora < 18) { console.log("Boa tarde!"); } else { console.log("Boa noite!"); } // Operador ternário let mensagem=(hora < 18) ? "Bom dia!" : "Boa noite!" ;

Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica.

// Declaração de função function saudacao(nome) { return "Olá, " + nome + "!"; } // Expressão de função const multiplicar = function(a, b) { return a * b; }; // Arrow function (ES6) const dividir = (a, b) => a / b; // Chamando funções let mensagem = saudacao("Ana"); // "Olá, Ana!" let produto = multiplicar(5, 3); // 15 let quociente = dividir(10, 2); // 5

Eventos

Eventos são ações que ocorrem no navegador, como cliques, movimentos do mouse, pressionamento de teclas, etc. Podemos usar JavaScript para responder a esses eventos.

// Métodos para adicionar eventos // 1. Atributo HTML // <button onclick="minhaFuncao()">Clique</button> // 2. Propriedade do elemento let btn = document.getElementById("meuBotao"); btn.onclick = function() { alert("Botão clicado!"); }; // 3. addEventListener (recomendado) btn.addEventListener("click", function() { alert("Botão clicado!"); });

DOM (Document Object Model)

O DOM é uma representação em forma de árvore da estrutura HTML de uma página. JavaScript pode manipular o DOM para alterar conteúdo, estilo e estrutura da página.

// Selecionando elementos let titulo = document.getElementById("meuTitulo"); let paragrafos = document.getElementsByClassName("meuParagrafo"); let elementos = document.querySelectorAll("div.minhaClasse"); // Alterando conteúdo titulo.innerHTML = "Novo Título"; titulo.textContent = "Outro Título"; // Alterando estilos titulo.style.color = "red"; titulo.style.fontSize = "24px"; // Adicionando/removendo classes titulo.classList.add("destaque"); titulo.classList.remove("antigo"); // Criando elementos let novoParagrafo = document.createElement("p"); novoParagrafo.textContent = "Este é um novo parágrafo"; document.body.appendChild(novoParagrafo);

Interação com APIs e Fetch

O que é uma API e JSON

API (Application Programming Interface) é um conjunto de regras que permite que diferentes softwares se comuniquem. JSON (JavaScript Object Notation) é um formato leve de troca de dados, fácil para humanos lerem e escreverem, e para máquinas parsearem e gerarem.

// Exemplo de JSON { "usuarios": [ { "id": 1, "nome": "João Silva", "email": "joao@exemplo.com" }, { "id": 2, "nome": "Maria Santos", "email": "maria@exemplo.com" } ] }

Como usar o fetch()

A API Fetch fornece uma interface para buscar recursos (incluindo através da rede). É uma alternativa mais poderosa e flexível ao XMLHttpRequest.

// Requisição GET simples fetch('https://api.exemplo.com/usuarios') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Erro:', error)); // Requisição POST fetch('https://api.exemplo.com/usuarios', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ nome: 'Novo Usuário', email: 'novo@exemplo.com' }) }) .then(response => response.json()) .then(data => console.log(data));
