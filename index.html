O que é JavaScript e como adicioná-lo ao HTML

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. É uma das três tecnologias principais da web, junto com HTML e CSS.

Como adicionar JavaScript ao HTML:

<!-- Método 1: Inline -->
<button onclick="alert('Olá Mundo!')">Clique aqui</button>

<!-- Método 2: Interno -->
<script>
function saudacao() {
  alert('Olá Mundo!');
}
</script>

<!-- Método 3: Externo (recomendado) -->
<script src="script.js"></script>

Exemplo Interativo:

Variáveis

Variáveis são containers para armazenar dados. Em JavaScript, podemos declarar variáveis usando var, let ou const.

Demonstração de Variáveis:

Código do Demo
<button onclick="demonstrateVariables()">Executar Demonstração</button>
            <div id="variables-output"></div>
function demonstrateVariables() {
              const output = document.getElementById('variables-output');
            
              var varExemplo = "Variável com var";
              let letExemplo = "Variável com let";
              const constExemplo = "Variável com const";
            
              letExemplo = "Novo valor para let";
            
              output.innerHTML = `
                <p><strong>var:</strong> ${varExemplo}</p>
                <p><strong>let:</strong> ${letExemplo}</p>
                <p><strong>const:</strong> ${constExemplo}</p>
                <p><em>Nota: Tentar reatribuir uma constante causaria um erro.</em></p>
              `.trim();
            }

Tipos de Dados

JavaScript tem tipos de dados dinâmicos, o que significa que uma mesma variável pode conter diferentes tipos de dados.

Verificar Tipo de Dado:

Código do Demo
<input type="text" id="data-type-input" placeholder="Digite qualquer valor">
            <button onclick="checkDataType()">Verificar Tipo</button>
            <p id="data-type-output"></p>
function checkDataType() {
              const rawValue = document.getElementById('data-type-input').value;
              const value = rawValue.trim();
              const output = document.getElementById('data-type-output');
            
              if (value === '') {
                output.innerHTML = 'Entrada vazia é do tipo: string';
                return;
              }
            
              if (!Number.isNaN(Number(value))) {
                const numberValue = Number(value);
                output.innerHTML = `${value} é do tipo: ${typeof numberValue}`;
                return;
              }
            
              const lowered = value.toLowerCase();
              if (lowered === 'true' || lowered === 'false') {
                const bool = lowered === 'true';
                output.innerHTML = `${value} é do tipo: ${typeof bool}`;
                return;
              }
            
              output.innerHTML = `${value} é do tipo: ${typeof value}`;
            }

Operadores e Condicionais

Operadores permitem realizar operações em variáveis e valores. Condicionais permitem executar código baseado em condições.

Operadores:

// Aritméticos
let soma = 10 + 5; // 15
let subtracao = 10 - 5; // 5
let multiplicacao = 10 * 5; // 50
let divisao = 10 / 5; // 2
let resto = 10 % 3; // 1

// Comparação
let igual = 5 == "5"; // true (valor igual)
let estritamenteIgual = 5 === "5"; // false (valor e tipo iguais)
let diferente = 5 != "5"; // false
let maiorQue = 10 > 5; // true

// Lógicos
let e = (5 > 3) && (10 > 5); // true
let ou = (5 > 10) || (10 > 5); // true

Condicionais:

// if/else
let hora = 14;

if (hora < 12) { console.log("Bom dia!"); } else if (hora < 18) { console.log("Boa tarde!"); } else { console.log("Boa
  noite!"); }

// Operador ternário
let mensagem = (hora < 18) ? "Bom dia!" : "Boa noite!";

Verificador de Idade:

Código do Demo
<input type="number" id="age-input" placeholder="Digite sua idade">
            <button onclick="checkAge()">Verificar</button>
            <p id="age-output"></p>
function checkAge() {
              const input = document.getElementById('age-input');
              const output = document.getElementById('age-output');
              const age = Number.parseInt(input.value, 10);
            
              if (Number.isNaN(age)) {
                output.innerHTML = 'Por favor, digite uma idade válida.';
                return;
              }
            
              if (age < 0) {
                output.innerHTML = 'Idade não pode ser negativa.';
              } else if (age < 13) {
                output.innerHTML = 'Você é uma criança.';
              } else if (age < 18) {
                output.innerHTML = 'Você é um adolescente.';
              } else if (age < 65) {
                output.innerHTML = 'Você é um adulto.';
              } else {
                output.innerHTML = 'Você é um idoso.';
              }
            }

Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica.

Calculadora Simples:

Código do Demo
<input type="number" id="num1" placeholder="Número 1">
            <input type="number" id="num2" placeholder="Número 2">
            <select id="operation">
              <option value="add">Somar</option>
              <option value="subtract">Subtrair</option>
              <option value="multiply">Multiplicar</option>
              <option value="divide">Dividir</option>
            </select>
            <button onclick="calculate()">Calcular</button>
            <p id="calculation-output"></p>
function calculate() {
              const num1 = Number.parseFloat(document.getElementById('num1').value);
              const num2 = Number.parseFloat(document.getElementById('num2').value);
              const operation = document.getElementById('operation').value;
              const output = document.getElementById('calculation-output');
            
              if (Number.isNaN(num1) || Number.isNaN(num2)) {
                output.innerHTML = 'Por favor, digite números válidos.';
                return;
              }
            
              let result;
              switch (operation) {
                case 'add':
                  result = num1 + num2;
                  output.innerHTML = `${num1} + ${num2} = ${result}`;
                  break;
                case 'subtract':
                  result = num1 - num2;
                  output.innerHTML = `${num1} - ${num2} = ${result}`;
                  break;
                case 'multiply':
                  result = num1 * num2;
                  output.innerHTML = `${num1} × ${num2} = ${result}`;
                  break;
                case 'divide':
                  if (num2 === 0) {
                    output.innerHTML = 'Erro: Divisão por zero!';
                    return;
                  }
                  result = num1 / num2;
                  output.innerHTML = `${num1} ÷ ${num2} = ${result}`;
                  break;
                default:
                  output.innerHTML = 'Operação desconhecida.';
              }
            }

Eventos

Eventos são ações que ocorrem no navegador, como cliques, movimentos do mouse, pressionamento de teclas, etc. Podemos usar JavaScript para responder a esses eventos.

Área de Eventos:

Passe o mouse aqui

Código do Demo
<button id="click-btn">Clique Aqui</button>
            <input type="text" id="key-input" placeholder="Digite algo aqui">
            <div id="mouse-area">Passe o mouse aqui</div>
            <p id="events-output"></p>
document.addEventListener('DOMContentLoaded', () => {
              const clickBtn = document.getElementById('click-btn');
              const keyInput = document.getElementById('key-input');
              const mouseArea = document.getElementById('mouse-area');
              const output = document.getElementById('events-output');
            
              clickBtn.addEventListener('click', () => {
                output.innerHTML = 'Botão foi clicado!';
              });
            
              keyInput.addEventListener('input', (event) => {
                output.innerHTML = `Você digitou: ${event.target.value}`;
              });
            
              mouseArea.addEventListener('mouseover', () => {
                mouseArea.style.backgroundColor = '#d4edda';
                output.innerHTML = 'Mouse sobre a área!';
              });
            
              mouseArea.addEventListener('mouseout', () => {
                mouseArea.style.backgroundColor = '#eee';
                output.innerHTML = 'Mouse saiu da área!';
              });
            });

DOM (Document Object Model)

O DOM é uma representação em forma de árvore da estrutura HTML de uma página. JavaScript pode manipular o DOM para alterar conteúdo, estilo e estrutura da página.

Manipulação do DOM:

Código do Demo
<input type="text" id="dom-input" placeholder="Digite um texto">
            <button onclick="addElement()">Adicionar Elemento</button>
            <button onclick="changeColor()">Mudar Cor</button>
            <button onclick="clearElements()">Limpar</button>
            <div id="dom-output"></div>
function addElement() {
              const input = document.getElementById('dom-input');
              const output = document.getElementById('dom-output');
              const value = input.value.trim();
            
              if (value === '') {
                alert('Por favor, digite algum texto.');
                return;
              }
            
              const newElement = document.createElement('div');
              newElement.textContent = value;
              newElement.classList.add('dom-item');
            
              output.appendChild(newElement);
              input.value = '';
            }
            
            function changeColor() {
              const output = document.getElementById('dom-output');
              const elements = output.querySelectorAll('.dom-item');
              const colors = ['#e8f4fd', '#fff3cd', '#d1ecf1', '#d4edda', '#f8d7da'];
            
              elements.forEach((element, index) => {
                element.style.backgroundColor = colors[index % colors.length];
              });
            }
            
            function clearElements() {
              document.getElementById('dom-output').innerHTML = '';
            }

Interação com APIs e Fetch

O que é uma API e JSON

API (Application Programming Interface) é um conjunto de regras que permite que diferentes softwares se comuniquem. JSON (JavaScript Object Notation) é um formato leve de troca de dados, fácil para humanos lerem e escreverem, e para máquinas parsearem e gerarem.

{
  "usuarios": [
  {
  "id": 1,
  "nome": "João Silva",
  "email": "joao@exemplo.com"
  },
  {
  "id": 2,
  "nome": "Maria Santos",
  "email": "maria@exemplo.com"
  }
  ]
  }

Como usar o fetch()

A API Fetch fornece uma interface para buscar recursos (incluindo através da rede). É uma alternativa mais poderosa e flexível ao XMLHttpRequest.

// Requisição GET simples
fetch('https://api.exemplo.com/usuarios')
  .then((response) => response.json())
  .then((data) => console.log(data))
  .catch((error) => console.error('Erro:', error));

// Requisição POST
fetch('https://api.exemplo.com/usuarios', {
  method: 'POST',
  headers: {
  'Content-Type': 'application/json'
  },
  body: JSON.stringify({
  nome: 'Novo Usuário',
  email: 'novo@exemplo.com'
  })
})
  .then((response) => response.json())
  .then((data) => console.log(data));

Exemplo prático com API pública

Vamos usar a JSONPlaceholder API, uma API REST fake para testes e prototipagem.

Buscar Dados da API:

Código do Demo
<select id="api-endpoint">
              <option value="posts">Posts</option>
              <option value="comments">Comentários</option>
              <option value="users">Usuários</option>
            </select>
            <button onclick="fetchData()">Buscar Dados</button>
            <button onclick="clearApiResult()">Limpar</button>
            <div id="api-result" class="api-result"></div>
function fetchData() {
              const endpoint = document.getElementById('api-endpoint').value;
              const apiUrl = `https://jsonplaceholder.typicode.com/${endpoint}`;
              const resultDiv = document.getElementById('api-result');
            
              resultDiv.innerHTML = 'Carregando...';
            
              fetch(apiUrl)
                .then((response) => {
                  if (!response.ok) {
                    throw new Error('Erro na requisição: ' + response.status);
                  }
                  return response.json();
                })
                .then((data) => {
                  const limitedData = data.slice(0, 5);
                  resultDiv.innerHTML = `
${JSON.stringify(limitedData, null, 2)}
`;
              })
              .catch((error) => {
              resultDiv.innerHTML = `Erro: ${error.message}`;
              });
              }
            
              function clearApiResult() {
              document.getElementById('api-result').innerHTML = '';
              }
